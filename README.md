# Excentoid Explorer

Interactive 3D explorer for **excentoids** — a new family of geometric shapes discovered by Zsák Zoltán, formed by ruled surfaces between eccentrically offset generator circles.

**[Live Demo](https://zebiv.com/excentoid/)**

## About Excentoids

Excentoids are 3D shapes generated by connecting closed curves (typically circles) placed in different planes with eccentrically offset centers. The surfaces between adjacent circles are **ruled surfaces** — every point on the surface lies on a straight line segment connecting the two generator curves.

The family includes:
- **Duploid** (2 circles) — the foundational shape, extending the oloid concept
- **Trioid** (3 circles) — inspired by Borromean chains
- **Quadroid** through **Nonaoid** (4–9 circles)
- **Anti-variants** — toroidal topology versions of each shape

Key parameter: **ψ = r/e** (eccentricity ratio), where r is the circle radius and e is the eccentric offset. The paper recommends ψ = √10 ≈ 3.162.

## Features

- WebGL 2 rendering with Phong shading
- All 8 excentoid types (Duploid through Nonaoid)
- Anti-variant toggle (toroidal topology)
- Adjustable eccentricity ratio (ψ)
- Solid, wireframe, and combined display modes
- Generator circle visualization
- Auto-rotation with mouse/touch orbit controls
- Live mathematical equations display

## Controls

- **Drag** — orbit camera
- **Shift+Drag** — pan
- **Scroll** — zoom
- **Touch** — single finger orbit, pinch zoom

## Tech

Vanilla HTML/CSS/JS, WebGL 2 (no libraries)
